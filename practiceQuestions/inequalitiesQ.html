Basic (1-7)

▶1. Solve for x: x + 3 > 7
Answer: x > 4
Step-by-step Explanation:
1. Start with the inequality: x + 3 > 7.
2. To get x alone, subtract 3 from both sides: x + 3 - 3 > 7 - 3.
3. This gives: x > 4.
4. So, any number greater than 4 is a solution.
▶2. Solve for y: 2y < 10
Answer: y < 5
Step-by-step Explanation:
1. Start with the inequality: 2y < 10.
2. Divide both sides by 2 to get y alone: 2y/2 < 10/2.
3. This gives: y < 5.
4. So, any number less than 5 is a solution.
▶3. Solve for z: z - 4 ≥ 2
Answer: z ≥ 6
Step-by-step Explanation:
1. Start with the inequality: z - 4 ≥ 2.
2. Add 4 to both sides: z - 4 + 4 ≥ 2 + 4.
3. This gives: z ≥ 6.
4. So, any number 6 or greater is a solution.
▶4. Solve for x: 5x ≤ 20
Answer: x ≤ 4
Step-by-step Explanation:
1. Start with the inequality: 5x ≤ 20.
2. Divide both sides by 5: 5x/5 ≤ 20/5.
3. This gives: x ≤ 4.
4. So, any number 4 or less is a solution.
▶5. Solve for y: -3y > 9
Answer: y < -3
Step-by-step Explanation:
1. Start with the inequality: -3y > 9.
2. Divide both sides by -3. Remember: when you divide by a negative, the inequality sign flips!
3. -3y/-3 < 9/-3 → y < -3.
4. So, any number less than -3 is a solution.
▶6. Solve for x: 4 < x + 2 ≤ 10
Answer: 2 < x ≤ 8
Step-by-step Explanation:
1. Start with the compound inequality: 4 < x + 2 ≤ 10.
2. Subtract 2 from all parts: 4 - 2 < x + 2 - 2 ≤ 10 - 2.
3. This gives: 2 < x ≤ 8.
4. So, x is greater than 2 and less than or equal to 8.
▶7. Solve for y: y/2 ≥ 3
Answer: y ≥ 6
Step-by-step Explanation:
1. Start with the inequality: y/2 ≥ 3.
2. Multiply both sides by 2: (y/2) × 2 ≥ 3 × 2.
3. This gives: y ≥ 6.
4. So, any number 6 or greater is a solution.

Moderate (8-14)

▶8. Solve for x: 2x - 5 < 9
Answer: x < 7
Step-by-step Explanation:
1. Start with the inequality: 2x - 5 < 9.
2. Add 5 to both sides: 2x - 5 + 5 < 9 + 5.
3. This gives: 2x < 14.
4. Divide both sides by 2: 2x/2 < 14/2 → x < 7.
▶9. Solve for y: 3y + 4 ≥ 10
Answer: y ≥ 2
Step-by-step Explanation:
1. Start with the inequality: 3y + 4 ≥ 10.
2. Subtract 4 from both sides: 3y + 4 - 4 ≥ 10 - 4.
3. This gives: 3y ≥ 6.
4. Divide both sides by 3: 3y/3 ≥ 6/3 → y ≥ 2.
▶10. Solve for z: 5z <= 2z + 12
Answer: z ≤ 4
Step-by-step Explanation:
1. Start with the inequality: 5z ≤ 2z + 12.
2. Subtract 2z from both sides: 5z - 2z ≤ 2z + 12 - 2z.
3. This gives: 3z ≤ 12.
4. Divide both sides by 3: 3z/3 ≤ 12/3 → z ≤ 4.
▶11. Solve for x: 7 - x > 2
Answer: x < 5
Step-by-step Explanation:
1. Start with the inequality: 7 - x > 2.
2. Subtract 7 from both sides: 7 - x - 7 > 2 - 7.
3. This gives: -x > -5.
4. Multiply both sides by -1 (flip the sign!): x < 5.
▶12. Solve for y: 4y + 8 > 0
Answer: y > -2
Step-by-step Explanation:
1. Start with the inequality: 4y + 8 > 0.
2. Subtract 8 from both sides: 4y + 8 - 8 > 0 - 8.
3. This gives: 4y > -8.
4. Divide both sides by 4: 4y/4 > -8/4 → y > -2.
▶13. Solve for x: 2(x - 1) ≥ 6
Answer: x ≥ 4
Step-by-step Explanation:
1. Start with the inequality: 2(x - 1) ≥ 6.
2. Divide both sides by 2: (x - 1) ≥ 3.
3. Add 1 to both sides: x - 1 + 1 ≥ 3 + 1 → x ≥ 4.
▶14. Solve for y: 3 - 2y < 7
Answer: y > -2
Step-by-step Explanation:
1. Start with the inequality: 3 - 2y < 7.
2. Subtract 3 from both sides: 3 - 2y - 3 < 7 - 3.
3. This gives: -2y < 4.
4. Divide both sides by -2 (flip the sign!): y > -2.

Advanced (15-20)

▶15. Solve for x: |x - 2| < 5
Answer: -3 < x < 7
Step-by-step Explanation:
1. Start with the absolute value inequality: |x - 2| < 5.
2. This means: -5 < x - 2 < 5.
3. Add 2 to all parts: -5 + 2 < x - 2 + 2 < 5 + 2.
4. This gives: -3 < x < 7.
▶16. Solve for y: |2y + 1| ≥ 7
Answer: y ≤ -4 or y ≥ 3
Step-by-step Explanation:
1. Start with the absolute value inequality: |2y + 1| ≥ 7.
2. This means: 2y + 1 ≤ -7 or 2y + 1 ≥ 7.
3. Solve each part:
- 2y + 1 ≤ -7 → 2y ≤ -8 → y ≤ -4
- 2y + 1 ≥ 7 → 2y ≥ 6 → y ≥ 3
4. So, y ≤ -4 or y ≥ 3.
▶17. Solve for z: |z + 5| <= 2
Answer: -7 ≤ z ≤ -3
Step-by-step Explanation:
1. Start with the absolute value inequality: |z + 5| ≤ 2.
2. This means: -2 ≤ z + 5 ≤ 2.
3. Subtract 5 from all parts: -2 - 5 ≤ z + 5 - 5 ≤ 2 - 5.
4. This gives: -7 ≤ z ≤ -3.
▶18. Solve for x: 3x - 4 > 2x + 5
Answer: x > 9
Step-by-step Explanation:
1. Start with the inequality: 3x - 4 > 2x + 5.
2. Subtract 2x from both sides: 3x - 2x - 4 > 2x - 2x + 5.
3. This gives: x - 4 > 5.
4. Add 4 to both sides: x - 4 + 4 > 5 + 4 → x > 9.
▶19. Solve for y: 2y + 3 < 5y - 6
Answer: y > 3
Step-by-step Explanation:
1. Start with the inequality: 2y + 3 < 5y - 6.
2. Subtract 2y from both sides: 2y + 3 - 2y < 5y - 6 - 2y.
3. This gives: 3 < 3y - 6.
4. Add 6 to both sides: 3 + 6 < 3y - 6 + 6.
5. This gives: 9 < 3y.
6. Divide both sides by 3: 9/3 < 3y/3 → 3 < y, or y > 3.
▶20. Solve for x: (x - 2)/(x + 1) > 0
Answer: x < -1 or x > 2
Step-by-step Explanation:
1. Start with the rational inequality: (x - 2)/(x + 1) > 0.
2. The expression is positive when both numerator and denominator are positive, or both are negative.
3. Case 1: x - 2 > 0 and x + 1 > 0 → x > 2 and x > -1 → x > 2.
4. Case 2: x - 2 < 0 and x + 1 < 0 → x < 2 and x < -1 → x < -1.
5. So, the solution is x < -1 or x > 2.
