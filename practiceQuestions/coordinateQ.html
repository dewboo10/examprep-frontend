â–¶1. Find the distance between points (3, 4) and (7, 7).
Answer: 5 units.
Explanation: Use the distance formula: \(\sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}\). Plug in the coordinates: \(x_1 = 3\), \(y_1 = 4\), \(x_2 = 7\), \(y_2 = 7\). Calculate: \(x_2 - x_1 = 7 - 3 = 4\), \(y_2 - y_1 = 7 - 4 = 3\). Distance = \(\sqrt{4^2 + 3^2} = \sqrt{16 + 9} = \sqrt{25} = 5\) units.
â–¶2. Find the midpoint of the line segment joining (2, 3) and (4, 5).
Answer: (3, 4).
Explanation: Use the midpoint formula: \(\left( \frac{x_1 + x_2}{2}, \frac{y_1 + y_2}{2} \right)\). Plug in: \(x_1 = 2\), \(y_1 = 3\), \(x_2 = 4\), \(y_2 = 5\). Calculate: \(x_{\text{mid}} = \frac{2 + 4}{2} = 3\), \(y_{\text{mid}} = \frac{3 + 5}{2} = 4\).
â–¶3. Find the slope of the line passing through (1, 2) and (3, 6).
